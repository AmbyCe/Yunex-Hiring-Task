# :car: Intersections CLI
Intersection management Java program - allows you to create, delete and list intersections.

It is controlled by arguments at startup and stores data in a JSON file.

## :wrench: Installation instructions
- Use [releases](https://github.com/AmbyCe/Yunex-Hiring-Task/releases/latest) or build using Maven `mvn clean package`.

## :keyboard: Commands
- `--list-items` - Lists all intersections that are in JSON file.
- `--create <ID> <NAME>` - Creates new intersection.
- `--delete <ID>` - Deletes an intersection.
